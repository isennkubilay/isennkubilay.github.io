Preventing Overfitting

When your model makes good predictions on the same data that was used to train it but shows poor results with data that hasn't seen before, we say that the model is overfitting.

Here are 7 ways you can deal with overfitting in Deep Learning neural networks.

  1. Train your model on more data

The more data you feed the model, the more likely it will start generalizing (instead of memorizing the training set.) Look at the relationship between dataset size and error.

2. Augment your dataset

You can automatically augment your dataset by transforming existing images in different ways to make the data more diverse. Some examples:

  • Zoom in/out
  • Contrast changes
  • Horizontal/vertical flips
  • Noise addition

3. Make your model simpler

  • Reduce the number of layers
  • Reduce the number of weights

The more complex your model is, the more capacity it has to memorize the dataset (hence, the easier it will overfit.)

Simplifying the model will force it to generalize.

4. Stop the learning process before overfitting This is known as "Early Stopping." Identify when overfitting starts happening and stop the learning process before it does. Plotting the training and validation errors will give you what you need for this.

5. Standardize input data. Smaller weights can result in a model less prone to overfit. Rescaling input data is a way to constraint these weights and keep them from increasing disproportionally.

6. Use Dropouts

Dropout is a regularization method that randomly ignores some of the outputs of a layer. This simulates the process of training different neural networks with different architectures in parallel, which is a way to avoid overfitting.

7. L1 and L2 regularization

These refer to a technique that penalizes the loss function to keep the weights of the network constrained.

This means that the network is forced to generalize better because it can't grow the weights without limit.

Precision Recall

We can think precision and recall as accuracy choices for different needs.

And extreme example would be: I live in an island inhabited by rabbits and poisonous snakes. Both population lives in holes. I can feed a week with a rabbit which is enough for rescue boats to find me. If a snake bites me, I die. According to the situation I only need to hunt 1 rabbit. With %100 precision. I don't need to try hard

Another example: Intel says. There is a atomic bomb in Ankara. The coordinates are unknown. What we know is the remote controller is at Kizilay metro station. We must find the carrier at all cost or the city of Ankara only exist in the pages of history. We decide to quarantine the metro station so remote doesn't come up to the surface where it can trigger the apocalypse. In other words we tagged all passengers as positive. We chose %100 recall.
